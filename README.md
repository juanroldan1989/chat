# Chat

Chat web application implemented via WebSockets using [faye-websocket](https://github.com/faye/faye-websocket-ruby) gem.

### Development

Clone repository:
```
git clone git@github.com:juanroldan1989/chat.git
```

Install gems:
```
bundle install
```

Run server:
```
foreman start
```
